# EventLens System Architecture

## Information Flow

News Sources

↓

Collection Layer

↓

Event Detection

↓

AI Processing

↓

Event Database

↓

Recommendation Engine

↓

Frontend

---

## Collection Layer

Responsible for collecting information from trusted sources.

---

## Event Detection

Groups multiple articles describing the same event into a single EventLens page.

---

## AI Processing

Extracts:

- Facts
- Claims
- Context
- Perspectives
- Unknowns

AI does not determine truth.

---

## Recommendation Engine

Learns user interests and reading behaviour.

---

## Frontend

Displays information in a structured, transparent and interactive manner.
