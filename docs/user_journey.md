### 5. Opening an Article

Goal:

Present the user with meaningful events instead of sensational headlines.

The home feed should feel calm, informative and personalized.

Every event card should contain:

- Event title
- One-line neutral summary
- Category
- Time published
- Reading time
- Confidence indicator (optional)
- Number of viewpoints available
- Timeline available
- Save button
